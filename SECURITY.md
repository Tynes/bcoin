# Security Policy

## Supported Versions

Versions of bcoin that are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0+   | :white_check_mark: |
| < 1.0.0-beta   | :x:         |

## Reporting a Vulnerability
